Desarrollo Web Full Stack

Programación Nivel: Intermedio

Aprende a crear aplicaciones web completas: HTML, CSS, JavaScript, backend y bases de datos.

Categoría
Programación
Nivel
Intermedio
Duración
60 horas
Volver al catálogo

Descripción general

En este curso aprenderás a construir una aplicación web de principio a fin, combinando frontend y backend. Practicarás con estructuras, componentes y buenas prácticas para desarrollar proyectos mantenibles y listos para producción.

Profesor/a responsable

Laura Martínez — Desarrolladora full stack y arquitecta de software.

Temario

  1. HTML semántico y estructura de una web moderna
  2. CSS y maquetación responsive
  3. JavaScript: DOM, eventos y validación
  4. Consumo de APIs y asincronía (fetch)
  5. Backend: rutas, controladores y lógica de negocio
  6. Persistencia: bases de datos y operaciones CRUD
  7. Buenas prácticas: seguridad, testing y despliegue

Requisitos previos

  • Conocimientos básicos de informática (uso de navegador y archivos)
  • Recomendado: nociones de HTML/CSS (no obligatorio)
  • Ganas de practicar: el curso es muy orientado a proyecto
