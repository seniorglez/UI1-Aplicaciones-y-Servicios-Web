Desarrollo Web Full Stack

Categoría: Tecnología

Nivel: Intermedio

Duración: 80 horas

Descripción general

Curso orientado a comprender el desarrollo de aplicaciones web modernas, abordando tanto el lado cliente como el servidor, junto con persistencia de datos y buenas prácticas de estructuración.

Profesor/a responsable

Laura Martínez — Especialista en desarrollo web y arquitectura de software.

Temario

  1. Fundamentos de la web y estructura de proyectos
  2. HTML semántico y buenas prácticas
  3. Introducción a CSS y maquetación (se verá en unidades posteriores)
  4. Programación en el navegador con JavaScript (unidades posteriores)
  5. Backend: rutas, controladores y persistencia
  6. Bases de datos: modelado y consultas
  7. Proyecto final: aplicación web completa

Requisitos previos

  • Conocimientos básicos de informática
  • Familiaridad con conceptos de programación (recomendado)
