Programación en Java

Categoría: Tecnología

Nivel: Básico

Duración: 50 horas

Descripción general

Curso orientado a adquirir una base sólida de programación con Java, comprendiendo la sintaxis del lenguaje, el uso de estructuras de control y el desarrollo de aplicaciones con un enfoque práctico. Se trabajarán conceptos de programación orientada a objetos y buenas prácticas para escribir código claro y mantenible.

Profesor/a responsable

Laura Martínez — Especialista en desarrollo web y arquitectura de software.

Temario

  1. Introducción a Java y configuración del entorno
  2. Tipos de datos, variables y operadores
  3. Estructuras de control: condicionales y bucles
  4. Métodos, parámetros y retorno de valores
  5. Programación orientada a objetos: clases y objetos
  6. Encapsulación, herencia y polimorfismo
  7. Excepciones y manejo de errores
  8. Colecciones básicas: listas y mapas
  9. Mini-proyecto final: aplicación de consola con POO

Requisitos previos

  • No se requieren conocimientos previos de programación
  • Se recomienda manejo básico del ordenador y archivos
  • Acceso a un equipo con Java instalado o posibilidad de instalarlo
